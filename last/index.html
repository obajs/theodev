                Structural and Semantic Elements

                <html>: The root element of an HTML 
                document, containing all other elements.
                <head>: Contains meta-information about
                 the HTML document (e.g., title, meta tags).
                <title>: Sets the title of the document,
                 displayed in the browser's title bar or tab.
                <body>: Contains the content of the HTML 
                document such as text, images, links, etc.
                <header>: Represents a container for 
                introductory content or navigational links 
                (can be repeated).
                <footer>: Defines a footer for the document
                 or section.
                <main>: Specifies the primary content of 
                the document.
                <section>: Defines a section of content, 
                typically with a heading.
                <article>: Represents an independent,
                 self-contained piece of content.
                <aside>: Defines content aside from the content it is 
                placed next to (like sidebars).
                <nav>: Contains navigation links for the
                document.
                <div>: Defines a division or section in 
                an HTML document, used for styling or grouping content.
                <span>: Used to group inline elements and 
                apply CSS styles.

                Text Elements

                <h1> to <h6>: Define HTML headings, where <h1>
                 is the highest level (most important), and <h6> is the lowest.
                <p>: Defines a paragraph.
                <blockquote>: Indicates that the enclosed 
                text is a block-level quotation.
                <pre>: Defines preformatted text, 
                preserving spaces and line breaks.
                <br>: Inserts a line break.
                <hr>: Represents a thematic break (like a
                 horizontal rule).
                <strong>: Defines strong text, typically bold.
                <em>: Defines emphasized text, typically italicized.
                <b>: Makes the enclosed text bold.
                <i>: Makes the enclosed text italic.
                <u>: Underlines the enclosed text.
                <small>: Defines smaller text (typically for 
                footnotes or disclaimers).
                <mark>: Highlights text.
                <sup>: Defines superscript text.
                <sub>: Defines subscript text.
                <code>: Defines a fragment of computer code.
                <kbd>: Defines keyboard input.
                <samp>: Defines sample output from a program.
                <var>: Represents a variable in a mathematical 
                expression or programming context.
                <abbr>: Defines an abbreviation or acronym.
                <cite>: Represents a reference to a creative work.

                List Elements
                 
                <ul>: Defines an unordered list.
                <ol>: Defines an ordered list.
                <li>: Defines a list item.
                <dl>: Defines a description list.
                <dt>: Defines a term in a description list.
                <dd>: Defines the description or definition
                 associated with a term in a description list.

                Media Elements

                <img>: Embeds an image in the document.
                <audio>: Embeds sound content (like music or 
                audio files).
                <video>: Embeds video content.
                <source>: Specifies multiple media resources for
                 elements like <audio> or <video>.
                <track>: Defines text tracks for <video> or
                 <audio>, like subtitles.
                <figure>: Specifies content (like images) that
                 is referenced from the main content.
                <figcaption>: Provides a caption for the <
                figure> element.
                <iframe>: Embeds another HTML document inside 
                the current one.

                Table Elements
                
                <table>: Defines a table.
                <tr>: Defines a table row.
                <td>: Defines a table data cell.
                <th>: Defines a table header cell.
                <thead>: Groups the header content in a table.
                <tbody>: Groups the body content in a table.
                <tfoot>: Groups the footer content in a table.
                <caption>: Provides a title for a table.
                <colgroup>: Specifies a group of columns in a table.
                <col>: Specifies column properties for each column in 
                a <colgroup>.

                Form Elements

                <form>: Defines an HTML form for user input.
                <input>: Defines an input field for user data.
                <textarea>: Defines a multiline text input control.
                <button>: Defines a clickable button.
                <select>: Creates a dropdown list.
                <option>: Defines an option in a dropdown list.
                <optgroup>: Groups options within a <select> 
                element.
                <label>: Defines a label for an <input> element.
                <fieldset>: Groups related elements in a form.
                <legend>: Provides a title for a <fieldset>.
                <datalist>: Specifies a list of pre-defined options 
                for input fields.
                <output>: Represents the result of a calculation.

                Interactive Elements

                <a>: Defines a hyperlink to link from one page to
                 another.
                <details>: Defines additional details that can be 
                viewed or hidden by the user.
                <summary>: Defines a summary or caption for the
                 <details> element.
                <dialog>: Defines a dialog box or window.
                <menu>: Defines a list or menu of commands.

                Scripting and Metadata

                <script>: Embeds or references JavaScript code.
                <noscript>: Defines alternative content for users
                 who do not support or have disabled JavaScript.
                <link>: Defines a relationship between the current 
                document and an external resource (like CSS).
                <meta>: Provides metadata about the HTML document, 
                like description, author, or character set.
                <style>: Embeds CSS styles within the HTML document.
                <base>: Specifies the base URL for all relative URLs 
                in the document.
                Deprecated/Obsolete Elements (Rarely Used)
                <acronym>: Used to define an acronym (deprecated, 
                replaced by <abbr>).
                <applet>: Used to embed Java applets (deprecated).
                <bgsound>: Used to play background sound (deprecated).
                <center>: Used to center-align text (deprecated).
                <font>: Used to define font size, color, and face 
                (deprecated).
                <frame>, <frameset>: Used for framesets (deprecated).

        <br> (Line Break):
        Inserts a line break in the text. Often used to 
        create a new line in a paragraph.
        
        <hr> (Horizontal Rule):
        Creates a thematic break or horizontal line, often 
        used to separate content.
        
        <img> (Image):
        Embeds an image in the document. The src attribute
         specifies the image source.
        
        <input> (Input Field):
        Defines an input field for user data in a form. It
         can have various types such as text, checkbox, radio, password, etc.
        
        <meta> (Metadata):
        Provides metadata about the HTML document, like 
        character encoding (charset), author, or description for search engines.
        
        <link> (Link to External Resources):
        Defines a relationship between the current document 
        and an external resource, typically used to link stylesheets (rel="stylesheet").
        
        <area> (Image Map Area):
        Defines an area within an image map, usually inside a
         <map> element, to create clickable areas on an image.
        
            <base> (Base URL):
        Specifies the base URL for all relative URLs in the document.
        
        <col> (Table Column):
        Specifies column properties for each column within
         a <colgroup> in a table.
        
            <embed> (Embedded Content):
        Embeds external content such as media files or
         interactive content (e.g., PDFs, Flash files).
        
        <param> (Object Parameters):
        Defines parameters for an object element, such as
         passing settings to a plugin or embedded media.
        
        <source> (Media Source):
        Specifies multiple media resources for elements like 
        <audio> or <video>. It allows specifying different 
        formats for different browsers.
        
            <track> (Text Track):
        Specifies text tracks for media elements (<video> 
        or <audio>), like subtitles or captions.
        
            <wbr> (Word Break Opportunity):
        Suggests a place in text where the browser can
         optionally break a word and wrap it to the next line.
        
        These elements are called "void elements" because 
        they cannot have any content or closing tags; they 
        are written as self-closing elements.

            
            

                
STEP3  the Attributes

            

            Global Attributes (Applicable to most HTML elements)
id: Specifies a unique id for an HTML element.

Example: <div id="main">
class: Specifies one or more class names for an element (used for CSS and JavaScript).

Example: <p class="intro">
style: Specifies inline CSS styles for an element.

Example: <h1 style="color:blue;">
title: Provides extra information about an element (usually
 shown as a tooltip when the mouse hovers over the element).

Example: <abbr title="World Health Organization">WHO</abbr>
*data- (Custom data attributes)**: Allows you to store custom data for an element. 
You define your own attributes using this prefix.

Example: <div data-user-id="123">
lang: Specifies the language of the element's content.

Example: <p lang="en">This is an English paragraph.</p>
dir: Specifies the text direction for the content (ltr for left-to-right, rtl for right-to-left).

Example: <p dir="rtl">This text is right-to-left.</p>
hidden: Hides an element from view.

Example: <p hidden>This text is hidden.</p>
tabindex: Specifies the tab order of an element (when the user presses the "Tab" key).

Example: <input type="text" tabindex="1">
contenteditable: Specifies whether the content of an element is editable or not.

Example: <div contenteditable="true">This text can be edited.</div>
draggable: Specifies whether an element is draggable or not.

Example: <img src="image.jpg" draggable="true">
spellcheck: Specifies whether the element should have its spelling and grammar checked or not.

Example: <textarea spellcheck="false"></textarea>
translate: Specifies whether the content of the element should be translated by the browser or not.

Example: <p translate="no">Do not translate this text.</p>
Form Attributes
type: Specifies the type of input element.

Example: <input type="text">
name: Specifies the name of an input element (used to reference form data after submission).

Example: <input type="text" name="username">
value: Specifies the value of an input element.

Example: vinput type="text" value="John">
placeholder: Provides a short hint that describes the expected value of an input field.

Example: <input type="text" placeholder="Enter your name">
disabled: Disables an element, making it unusable.

Example: vbutton disabled>Click Mev/button>
readonly: Specifies that an input field is read-only.

Example: <input type="text" value="John" readonly>
required: Specifies that an input field must be filled out before submitting the form.

Example: <input type="text" required>
min: Specifies the minimum value for an input field.

Example: <input type="number" min="1">
max: Specifies the maximum value for an input field.

Example: <input type="number" max="100">
pattern: Specifies a regular expression that the input element's value must match.

Example: <input type="text" pattern="[A-Za-z]{3,}">
autofocus: Automatically focuses on the element when the page loads.

Example: <input type="text" autofocus>
autocomplete: Specifies whether a form or input element should have autocomplete enabled.

Example: <input type="text" autocomplete="on">
multiple: Allows the user to select multiple values for certain input types.

Example: <input type="file" multiple>
Media Attributes
src: Specifies the URL of the media resource (image, video, audio).

Example: <img src="image.jpg">
alt: Provides alternative text for images (for accessibility and SEO).

Example: <img src="image.jpg" alt="A beautiful scenery">
width: Specifies the width of an image, video, or canvas.

Example: <img src="image.jpg" width="500">
height: Specifies the height of an image, video, or canvas.

Example: <img src="image.jpg" height="300">
controls: Displays media controls (for video or audio elements).

Example: <video src="video.mp4" controls></video>
autoplay: Specifies that the media should start playing as soon as it is ready.

Example: <audio src="audio.mp3" autoplay></audio>
loop: Specifies that the media should start over again when it ends.

Example: <video src="video.mp4" loop></video>
muted: Specifies that the media element should be muted.

Example: <video src="video.mp4" muted></video>
poster: Specifies an image to be shown while the video is downloading or until the user clicks play.

Example: <video src="video.mp4" poster="poster.jpg"></video>
Anchor (<a>) Tag Attributes
href: Specifies the URL of the page the link goes to.

Example: <a href="https://example.com">Visit Example</a>
target: Specifies where to open the linked document (_blank for new tab, _self for the same window).

Example: <a href="https://example.com" target="_blank">Open in new tab</a>
download: Specifies that the target will be downloaded when the link is clicked.

Example: <a href="file.pdf" download>Download PDF</a>
rel: Specifies the relationship between the current 
document and the linked document (e.g., "noopener",
 "nofollow").

Example: <a href="https://example.com" rel="noopener">
Visit Example</a>
Button and Input Attributes
type: Defines the button or input type (button, submit,
 reset).

Example: <button type="submit">Submit</button>
name: Specifies the name of the button or input field.

Example: <input type="text" name="username">
value: Specifies the value for buttons and input fields.

Example: <input type="submit" value="Submit">
Other Specialized Attributes
srcset: Specifies a list of image sources for 
different device screen sizes or pixel densities.

Example: <img src="image.jpg" srcset="image-2x.jpg
 2x, image-3x.jpg 3x">
defer: Specifies that the script should be executed 
after the document has been fully parsed.

Example: <script src="script.js" defer></script>
async: Specifies that the script should be executed
 asynchronously.

Example: <script src="script.js" async></script>
charset: Specifies the character encoding for HTML 
documents or external resources (e.g., in <meta> or <script>).

Example: <meta charset="UTF-8">
http-equiv: Provides an HTTP header for the content-type or refresh directive.

Example: <meta http-equiv="refresh" content="30">
sandbox: Applies restrictions to an iframe's content.

Example: <iframe src="page.html" sandbox></iframe>
srcdoc: Specifies the HTML content for an iframe.

Example: <iframe srcdoc="<p>This is an iframe content.</p>"></iframe>
form: Associates an input or button with a specific form.

Example: <button type="submit" form="myForm">Submit</button>
This list covers the most commonly used HTML attributes.

Definition of CSS and how to code in CSS Definition of CSS 
                CSS (Cascading Style Sheets) is a 
                stylesheet language used to describe 
                the presentation (look and formatting) 
                of a document written in HTML or XML. 
                It allows developers to control the 
                layout, design, and style of web pages, 
                separating content from presentation.

                With css you can:
                
    
                
  1. Control the color, fonts, spacing, and size of elements.
    2. 
                
  2. Define layout styles, such as grids or flexible designs for responsiveness.
    3. 
                
  3. Apply animations, transitions, and hover effects.
    4. 
                


                How to Code in CSS:
                1. Inline CSS: Written directly inside an HTML element using the style attribute.

                2. Internal CSS: Defined in a <style> block within the <head> section of an HTML file.

                3. External CSS: Written in a separate .css file and linked to an HTML document.


                Example of Each Type of CSS:

                1. Inline CSS:
                
                    <p style="color: blue; font-size: 20px;">This is a styled paragraph.</p>
